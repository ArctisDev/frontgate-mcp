Crie uma landing page completa para um SaaS de deploy e infraestrutura chamado Arctis Deploy.

Regras de execução:
- Use apenas o MCP chamado Frontgate.
- Não use nenhuma skill disponível.
- Não use nenhum outro MCP além do Frontgate.
- Toda a construção visual, estrutural e criativa deve se apoiar exclusivamente no Frontgate.

Objetivo:
Criar uma landing page com qualidade visual alta, identidade marcante e acabamento premium, explorando ao máximo a capacidade do MCP Frontgate.

Direção visual:
- premium minimalista
- forte senso de produto técnico
- visual mais proprietário e menos template
- sem terminal fake no hero
- sem excesso de cards genéricos
- sem poluição visual
- azul profundo como base da identidade
- dark theme elegante com contraste bem resolvido
- composição sólida, equilibrada e com boa densidade visual

Seções desejadas:
- hero com headline forte e subheadline clara
- seção de diferenciais estratégicos
- seção explicando o fluxo do produto
- recursos principais
- seção de posicionamento contra soluções engessadas
- CTA final
- footer

Contexto do produto:
Arctis Deploy é uma plataforma para desenvolvedores que querem fazer deploy com mais controle, menos lock-in e melhor racional de custo.

Tom da copy:
- técnico
- direto
- convincente
- maduro
- sem marketing inflado

Entregue:
- proposta completa da landing page
- estrutura de conteúdo
- copy de todas as seções
- direção de layout
- sistema visual sugerido
- componentes e blocos ideais
- justificativas de design quando necessário

Prioridade:
Quero que você explore o Frontgate como principal motor de design e composição dessa landing page.
