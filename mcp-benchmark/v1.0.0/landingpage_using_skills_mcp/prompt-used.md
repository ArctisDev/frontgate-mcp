Crie uma landing page completa para um SaaS de deploy e infraestrutura chamado Arctis Deploy.

Regras de execução:
- Use as skills disponíveis que fizerem sentido.
- Use o MCP Frontgate como MCP principal de design.
- Pode usar MCPs adicionais somente se forem realmente necessários para melhorar a execução.
- Não use ferramentas extras sem necessidade real.
- Priorize consistência visual, profundidade de layout e qualidade de decisão.

Objetivo:
Produzir uma landing page premium, minimalista e estrategicamente bem construída, combinando o melhor das skills disponíveis com o MCP Frontgate e MCPs complementares quando agregarem valor real.

Direção visual:
- visual premium com identidade própria
- aparência de software sério e bem posicionado
- composição refinada
- dark theme sofisticado
- azul profundo como base
- sem terminal ilustrativo no hero
- sem visual genérico de template SaaS
- sem excesso de badges, grids decorativos ou elementos clichês
- foco em clareza, autoridade e percepção de produto robusto

Seções desejadas:
- hero com proposta de valor forte e CTA claro
- prova de valor / credibilidade
- benefícios principais
- como funciona
- recursos de destaque
- diferenciação competitiva
- CTA final
- footer enxuto

Contexto do produto:
Arctis Deploy é uma plataforma de deploy para desenvolvedores que querem mais controle de infraestrutura, menos lock-in e custo mais sustentável conforme escalam.

Tom da copy:
- direto
- técnico
- seguro
- sem exageros
- sem clichês de marketing

Entregue:
- landing page completa
- estrutura de seções
- copy de cada parte
- proposta visual
- layout sugerido
- componentes recomendados
- microinterações
- racional das escolhas mais importantes

Prioridades:
- usar o Frontgate como peça central do design
- complementar com skills e outros MCPs apenas quando isso elevar claramente a qualidade do resultado
- maximizar qualidade visual, coerência e utilidade prática
