Crie uma landing page completa para um SaaS de deploy e infraestrutura chamado Arctis Deploy.

Regras de execução:
- Não use nenhuma skill disponível.
- Não use nenhum MCP.
- Trabalhe apenas com seu conhecimento interno e gere a solução diretamente.

Objetivo:
Criar uma landing page com estética premium, minimalista, forte identidade visual e aparência de produto técnico robusto, sem cair em visual genérico de template.

Direção visual:
- visual premium e limpo
- aparência moderna, mas sóbria
- design com sensação de engenharia séria e produto confiável
- evitar excesso de elementos decorativos
- evitar visual genérico de startup
- não usar mockup de terminal hero
- não usar dashboard falso exagerado
- foco em hierarquia visual forte, tipografia boa e blocos bem espaçados
- paleta puxada para azul profundo e tons escuros elegantes

Seções desejadas:
- hero com proposta de valor forte
- faixa de prova de valor ou trust
- benefícios principais
- como funciona
- recursos principais
- comparativo de posicionamento
- CTA final
- footer simples

Contexto do produto:
Arctis Deploy é uma plataforma para deploy de aplicações com mais controle de infraestrutura, menos lock-in e custo mais racional para desenvolvedores.

Tom da copy:
- direto
- técnico
- confiante
- sem hype vazio
- sem frases clichês tipo “revolucione seu workflow”

Entregue:
- estrutura completa da landing page
- copy principal de cada seção
- sugestão de layout
- direção visual detalhada
- componentes recomendados
- sugestões de microinterações
